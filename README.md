## Overview

IP-Trace-2026 is a high-performance geolocation dashboard designed for real-time network vector analysis. It leverages multiple intelligence APIs to provide comprehensive data on target IP addresses or geographical coordinates, presented in a premium "cyber-operations" aesthetic.

## Key Features

*   **Dual Tracking Modes**: Seamlessly switch between IP Interception and Satellite Coordinate Triangulation.
*   **Geospatial Intelligence**: Real-time map integration with inverted thermal styling for enhanced visibility.
*   **Deep Signal Analysis**: Comprehensive data retrieval including ISP, ASN, Proxy/VPN detection, and country-level intelligence.
*   **Full Trace Modal**: Detailed technical reports with raw JSON payload visualization.
*   **Nexus Shadow Aesthetics**: Premium dark-mode UI with glassmorphism, pixelated background animations, and responsive HUD elements.
*   **Instant Intelligence Export**: One-click signal data copying in a formatted intelligence report format.

## Technology Stack

*   **Frontend**: React 18 with TypeScript for strict type safety.
*   **Styling**: Vanilla CSS and Tailwind CSS for a modern, responsive UI.
*   **Icons**: Lucide React for consistent, high-quality vector iconography.
*   **Mapping**: Integrated Map Display with custom filter layering.
*   **Build Tool**: Vite for lightning-fast development and optimized production bundles.

## Getting Started

### Prerequisites

*   Node.js (v18 or higher)
*   npm or yarn

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/yourusername/IP-Trace-2026.git
    ```

2.  Install dependencies:
    ```bash
    npm install
    ```

3.  Set up environment variables:
    Create a `.env` file in the root directory and add your API keys:
    ```env
    VITE_IP_API_KEY=your_key_here
    ```

4.  Start the development server:
    ```bash
    npm run dev
    ```
